E-Commerce Demo

Progetto client-server in HTML, CSS, Vanilla JavaScript e Node.js.

La demo prevede due ruoli distinti: utente e admin.

Funzionalità principali

  • Visualizzazione catalogo prodotti
  • Acquisto con controllo crediti e stock
  • Dashboard admin per prodotti e crediti
  • Controlli lato server su sicurezza e validazione
